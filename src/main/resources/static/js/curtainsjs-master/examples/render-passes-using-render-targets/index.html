We are using two shader passes with 2 different render targets to post-process 2 sets of planes (deformation for the first set, rgb shift for the second set), and then one more shader pass to the whole scene to add vertical blur on scroll.

This illustrates how shader passes can be applied to only a specific set of planes or the whole scene and how you can add them.
